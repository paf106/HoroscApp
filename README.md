# HoroscApp

## Description
Aplicación para consultar el horóscopo del día.
Hecha a partir de un curso de Android de Aristidevs.

## Contenido
Esta aplicación contiene:
- Arquitectura MVVM
- Retrofit
- ViewBinding
- Navigation Component
- Material Design
- Animaciones
- Bottom Navigation
- RecyclerView
- ViewModels
- Flows
- Dagger Hilt
- Testing
